يرجى تطبيق التعديلات والإصلاحات التالية بالكامل مع اختبار كل نقطة بعد التنفيذ والتأكد من عملها بشكل صحيح داخل الموقع ولوحة تحكم الأدمن وقاعدة بيانات Supabase.

أولاً: إكمال الملف الشخصي للمستخدم

1- شاشة إكمال الملف الشخصي

عند تسجيل مستخدم جديد ودخوله إلى الموقع ثم انتقاله إلى صفحة:
"الملف الشخصي"

إذا لم يكن قد أكمل بياناته الشخصية:

- اعرض له لوحة / مودال إكمال الملف الشخصي أولاً.
- امنع إخفاء اللوحة حتى يتم إدخال البيانات الأساسية المطلوبة.

ويجب أن تحتوي على:

- رسالة ترحيبية احترافية.
- عبارة توضح أن:
  "تبقت خطوة أخيرة لإكمال إعداد حسابك وبياناتك الشخصية"

مع تصميم حديث ومتناسق مع تصميم الموقع.

ثانياً: البيانات الشخصية وربط Supabase

2- تعديل صفحة البيانات الشخصية

في صفحة:
"الملف الشخصي"

داخل أيقونة:
"البيانات الشخصية"

حالياً عند الضغط عليها:

- يتم فتح الحقول للتعديل مباشرة.

المطلوب:

- عند الضغط على الكرت يتم فقط عرض البيانات.
- لا يتم تفعيل التعديل إلا بعد الضغط على زر:
  "تعديل"

3- إصلاح حفظ البيانات وربط قاعدة البيانات

حالياً:

- البيانات لا تحفظ بشكل صحيح.
- البيانات لا تظهر داخل لوحة تحكم الأدمن.

قم بتنفيذ التالي:

أ) ربط Supabase بشكل كامل

- ربط جميع بيانات الموقع مع قاعدة بيانات Supabase.
- إنشاء الجداول الناقصة المطلوبة.
- تنظيم العلاقات بين الجداول بشكل احترافي.

ب) تخزين بيانات المستخدم كاملة

قم بحفظ جميع بيانات المستخدم كاملة داخل Supabase، بما يشمل:

- الاسم
- اسم الأب
- اللقب
- البريد الإلكتروني
- رقم الهاتف
- الجنس
- تاريخ الميلاد
- المحافظة
- المدينة
- العنوان الكامل
- الصورة الشخصية
- الموقع الجغرافي
- نوع تسجيل الدخول
- مزود تسجيل الدخول
- الجلسات
- الأجهزة
- كلمات المرور إذا كانت مستخدمة بنظام البريد وكلمة المرور
- سجل الإشعارات
- حالة الحساب
- الحظر والتقييد
- التوثيق
- النشاطات داخل التطبيق

ج) مزامنة لوحة الأدمن

- عرض المعلومات الأساسية والكاملة داخل لوحة تحكم الأدمن.
- تحديث البيانات مباشرة بعد تعديلها.
- التأكد من عدم فقدان أي بيانات.

ثالثاً: العملات والأسعار

4- إنشاء صفحات العملات والفئات

لم يتم:

- إنشاء صفحات العملات والمعادن والفئات التي تم إرسال رموزها سابقاً.
- جلب الأسعار من MetalPrice API.

المطلوب:

- إنشاء جميع الصفحات الناقصة تلقائياً.
- إنشاء أيقونات العملات والمعادن والفئات بنفس تصميم الموقع الحالي.
- إنشاء صفحات التفاصيل الخاصة بكل عملة أو معدن.

5- ربط الأسعار بـ MetalPrice API

قم بجلب الأسعار من:
MetalPrice API

وتأكد من:

- حذف مكونات الـ API القديمة من لوحة الأدمن > صفحة النظام.
- استخدام النظام الجديد فقط.
- تحديث الأسعار كل 8 ساعات فقط.
- حفظ الأسعار داخل قاعدة البيانات أو الكاش لتقليل الطلبات.

رابعاً: الأداء والاستقرار

6- إصلاح مشكلة إعادة تحميل الموقع

ما زالت مشكلة إعادة تحميل الموقع موجودة.

المشكلة:

- عند الخروج من التطبيق أو المتصفح لعدة ثوانٍ ثم العودة يتم إعادة تحميل الموقع بالكامل.

المطلوب:

- الحفاظ على حالة الصفحة الحالية.
- منع إعادة تحميل الصفحة بدون سبب.
- الحفاظ على الجلسة والبيانات المؤقتة.
- تحسين إدارة الـ Session والـ Cache والـ State Management.

خامساً: شارات التوثيق

7- تحسين شارة Legendary

اجعل شارة:
"Legendary"

كما هي بنفس التصميم الحالي لكن:

- أضف توهجاً خفيفاً ومتدرجاً خلف الشارة.
- اجعل التوهج سلساً وفخماً.
- لا تبالغ بقوة الإضاءة.
- حافظ على الهوية الحالية للشارة.

8- تحسين شارة Cyberpunk

قم بتحسين شارة:
"Cyberpunk"

بحيث:

- يصبح التوهج أقوى قليلاً.
- تصبح الحلقات الدوارة بنفس عدد حلقات شارة Legendary.
- تصبح الحركة أكثر سلاسة وحداثة.

سادساً: مراجعة الموقع بالكامل

9- مراجعة وتطوير شامل

قم بمراجعة الموقع بالكامل والبحث عن:

- الميزات الناقصة.
- الأخطاء البرمجية.
- المشاكل البصرية.
- الصفحات غير المكتملة.
- العناصر التي تحتاج تحسين تصميم.
- الأيقونات التي تحتاج إعادة تصميم.
- الكروت التي تحتاج تحسين توزيع وتنسيق.

وقم بتطوير وتحسين:

- الأداء
- التصميم
- تجربة المستخدم
- سرعة التنقل
- تناسق الواجهات

10- مراجعة الوضع المظلم والفاتح

راجع الموقع بالكامل في:

- الوضع المظلم
- الوضع الفاتح

وتأكد من:

- وضوح جميع النصوص.
- تناسق الألوان.
- عدم وجود نصوص باهتة.
- وضوح الأزرار والأيقونات.
- تناسق الخلفيات والكروت والحدود.
- ظهور جميع العناصر بشكل احترافي في كلا الوضعين.
