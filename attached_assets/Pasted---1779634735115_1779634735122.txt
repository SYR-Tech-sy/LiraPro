يرجى قراءة جميع النقاط التالية بعناية وتنفيذها بالكامل مع اختبار كل نقطة عملياً بعد التنفيذ والتأكد من أنها تعمل بدون أخطاء أو تعليق.

أولاً: مودال إكمال الملف الشخصي

1- إعادة الصورة القديمة

داخل مودال:
"إكمال الملف الشخصي"

أعد الصورة القديمة التي كانت تظهر سابقاً داخل أيقونة الصورة الشخصية.

لا تستخدم الصورة الحالية.

2- إصلاح تصميم وعمل المودال

حالياً المودال:

- غير سلس.
- يعلق أثناء الاستخدام.
- بعض الحقول لا تستجيب.
- قوائم الاختيار لا تعمل بشكل صحيح.

المطلوب:

- إعادة تصميم المودال بشكل حديث وسلس.
- إصلاح التعليق والبطء.
- إصلاح جميع قوائم الاختيار Select.
- التأكد من أن جميع الحقول قابلة للاستخدام بشكل طبيعي.
- التأكد من أن المودال يعمل بشكل ممتاز على الهاتف والكمبيوتر.

وأضف أيضاً داخل مودال:
"إكمال الملف الشخصي"

زر إضافي باسم:
"العودة إلى الصفحة الرئيسية"

وظيفته:

- يسمح للمستخدم بتجاهل إكمال الملف الشخصي مؤقتاً.
- عند الضغط عليه يتم إغلاق المودال والعودة للصفحة الرئيسية.

لكن:

- يبقى المودال يظهر لاحقاً عند دخول المستخدم مرة أخرى إلى صفحة الملف الشخصي طالما لم يكمل بياناته الأساسية.

ويجب:

- أن يكون الزر بتصميم متناسق مع تصميم المودال.
- مع الحفاظ على زر:
  "إكمال البيانات" أو "حفظ"
  كخيار أساسي وأوضح للمستخدم.

ثانياً: لوحة تحكم الأدمن - صفحة المستخدمين

3- إصلاح مودال تعديل بيانات المستخدم

في لوحة تحكم الأدمن:
صفحة:
"المستخدمين"

عند فتح مستخدم محدد ثم الضغط على:
"تعديل البيانات"

حالياً:

- لا تظهر جميع الحقول.
- لا تظهر قوائم الاختيار كاملة.

المطلوب:

- اجعل نموذج التعديل يعرض جميع الحقول بالكامل.
- اجعله بنفس تصميم صفحة:
  "إكمال تسجيل البيانات"

ويجب أن يتضمن:

- الاسم
- اسم الأب
- اللقب
- البريد الإلكتروني
- الهاتف
- الجنس
- تاريخ الميلاد
- المحافظة
- المدينة
- العنوان
- الصورة
- نوع تسجيل الدخول
- حالة الحساب
- الجلسات
- الأجهزة
- وجميع الحقول الأخرى

وأضف أيضاً:

- حقل عرض وتعديل:
  Supabase UID

ثالثاً: أسعار API العالمية

4- إصلاح جلب الأسعار

الموقع حالياً لا يجلب الأسعار من الـ API العالمي المخصص للأسعار.

المطلوب:

- التأكد من أن النظام يجلب الأسعار فعلياً من:
  MetalPrice API
- التأكد من تحديث العملات والمعادن بشكل صحيح.
- التأكد من ظهور البيانات داخل الموقع.
- إزالة أي API قديم أو غير مستخدم.

رابعاً: لوحة الأدمن - صفحة التجار

5- إعادة تصميم صفحة التجار

في لوحة تحكم الأدمن:
صفحة:
"التجار"

أعد تصميم بطاقات التجار بشكل حديث ومرتب.

ويجب أن تعرض البطاقة:

- جميع بيانات التاجر.
- صورة الشركة.
- الفئة.
- المدينة.
- حالة الحساب.
- التوثيق.
- عدد الأسعار.
- آخر نشاط.

وعند فتح التاجر:

- اعرض جميع الأسعار التي أضافها.
- اعرض جميع السلع والفئات التابعة له.
- اعرض زمن تحديث كل سعر.

وأضف:

- زر تعديل السعر.
- زر حذف السعر.
- تحديث البيانات مباشرة بعد التعديل أو الحذف.

خامساً: الأسعار المشبوهة

6- إضافة صفحة جديدة باسم:

"مشبوه"

داخل الشريط العلوي في لوحة تحكم الأدمن.

هذه الصفحة يجب أن:

- تعرض الأسعار الشاذة أو غير الطبيعية.
- تقارن السعر بأسعار باقي التجار من نفس الفئة.

مثال:
إذا كان جميع التجار يضعون سعر قريب من:
100

وأحدهم وضع:
500

يعتبر السعر:
مشبوهاً.

المطلوب:

- عدم نشر السعر المشبوه مباشرة.
- تحويله تلقائياً لصفحة:
  "مشبوه"

حتى يراجعه الأدمن.

ويجب أن تعرض الصفحة:

- بيانات التاجر كاملة.
- السعر المدخل.
- متوسط الأسعار الطبيعية.
- وقت الإضافة.

مع الأزرار التالية:

- قبول ونشر السعر.
- رفض السعر.

سادساً: صفحة طلب عضوية الإعلان عن الأسعار

7- تعديل كرت:

"انضم كمورد موثوق"

أ) تعديل النص

العبارة:
"على منصة LiraPro"

اجعلها:

- في المنتصف.
- متناسقة مع العبارة التي فوقها.
- بتوزيع وتصميم احترافي.

ب) عبارة مثال عرض المصدر

العبارة:
"مثال على عرض مصدر السعر"

اجعل:

- لون النص ذهبي.
- مع تأثير لمعان متحرك ناعم من اليسار.

ج) صورة المثال

اجعل صورة المثال:

- مطابقة تماماً للتصميم الحقيقي الموجود في:
  كرت المصدر داخل صفحة تفاصيل السلعة.
- بنفس الألوان.
- بنفس تنسيق اسم الشركة.
- مع اسم المصدر أعلى الكرت.

سابعاً: كرت الموثوقية

8- تعديل رقم الموثوقية

في كرت:
"كيف تزداد موثوقيتك"

غيّر الرقم:
95

إلى:
100

واجعل:

- الرقم باللون الذهبي.
- مع لمعان سلس متحرك من اليسار.

أما الأرقام التي قبله:

- اجعل ألوانها أفتح تدريجياً منه.

ثامناً: الأسئلة الشائعة

9- إضافة صفحة:

"الأسئلة الشائعة"

داخل القائمة الجانبية أسفل:
"سياسة الخصوصية"

ويجب:

- إنشاء أسئلة وأجوبة احترافية.
- تكون مرتبطة بالموقع فقط.
- تشرح:
  كيفية الاستخدام
  التوثيق
  الأسعار
  الإشعارات
  الحسابات
  التجار
  الحظر
  الأمان
  الخصوصية
  وغيرها.

مع تصميم احترافي ومتناسق مع الموقع.

أخيراً

يرجى التأكد من:

- تنفيذ جميع النقاط بالكامل.
- عدم تجاهل أي نقطة.
- اختبار جميع الميزات عملياً بعد التنفيذ.
- التأكد من عدم وجود أخطاء أو تعليق أو مشاكل تصميم.
